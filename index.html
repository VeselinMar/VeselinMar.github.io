    TournamentManager

    TournamentManager is a web application designed to simplify the organization and execution of football tournaments. What began as a personal solution grew out of my role in helping organize an annual football competition, where scoring, calculating points, and publishing results were still handled manually. Wanting to automate this process and reduce repetitive administrative work, I set out to build a tool that automates most of the tasks and provides live information about the state of the tournament to the interested parties. Through a guided wizard, organizers can create tournament pages, generate schedules, and manage teams with just a few clicks. The system currently supports the round-robin format, automatically tracking scores, calculating standings and possiblity to add sponsor banners/logos. Built with Django for backend logic and HTML5/CSS + vanilla JS for the frontend, the project emphasizes accessibility and ease of use, making it approachable for both organizers and participants.

    Mama-To-Be

    Mama-To-Be is a personal web application created for my wife, designed to serve as a dedicated blog with additional interactive features. It provides a comfortable space to write, share experiences, and manage content easily. Articles are created using a user-friendly TinyMCE editor, offering a word-processor-like writing experience. The application uses Django for backend logic and HTML/CSS/JavaScript with Bootstrap for the frontend. Originally built with PostgreSQL, deployment constraints on Render led to a switch to SQLite, with persistent data seeded from a JSON file during startup to keep hosting cost-free and maintenance simple.
