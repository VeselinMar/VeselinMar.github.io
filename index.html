    Mama-To-Be

    Mama-To-Be is a web application designed for mothers to share experiences and connect through a blog-style interface. Users can create articles using a word-processor-like editor powered by TinyMCE, making content creation intuitive. Initially developed with PostgreSQL, deployment constraints on Render led to a SQLite backend, with data persistence managed via a JSON file that seeds the database on startup. The stack combines Django for backend logic and HTML/CSS/JS and Bootstrap for the frontend. The project emphasizes cost-free hosting and maintainability, offering a simple, interactive platform for mothers to exchange knowledge and support.

    TournamentManager

    TournamentManager is a web application designed to simplify the organization and execution of football tournaments. What began as a personal solution grew out of my role in helping organize a small annual football competition, where scoring, calculating points, and publishing results were still handled manually. Wanting to automate this process and reduce repetitive administrative work, I set out to build a tool that makes tournament management faster, more transparent, and more engaging for everyone involved. Through a guided wizard, organizers can create tournament pages, generate schedules, and manage teams with just a few clicks. The system currently supports the round-robin format, automatically tracking scores and calculating standings. Built with Django for backend logic and HTML5/CSS for the frontend, the project emphasizes accessibility and ease of use, making it approachable for both organizers and participants. I am now extending the functionality to include a dedicated display panel, allowing fans to follow matches in semi-real time (with results entered manually after each game). This view also gives organizers the option to showcase sponsor content in a subtle, non-intrusive way, creating new opportunities for community engagement and support.
