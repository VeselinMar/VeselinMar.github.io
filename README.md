# Personal Portfolio Website

This repository contains the source code for my personal portfolio website, hosted via **GitHub Pages**.

Live Site: https://veselinmar.github.io/

---

## Project Structure

```
.
├── index.html          # Main HTML structure
├── style.css           # Main styling
├── mediaqueries.css    # Responsive design rules
└── assets/             # Images and other media (if applicable)
```

---

## Built With

- **HTML5** – Website structure
- **CSS3** – Styling and layout
- **Media Queries** – Responsive design
- **GitHub Pages** – Deployment & hosting

---

## Features

- Responsive design for desktop, tablet, and mobile
- Clean and modern UI
- Smooth layout and organized sections
- Optimized structure and styling separation

---

## Responsive Design

Responsive behavior is handled in:

```
mediaqueries.css
```

Breakpoints are defined to ensure proper layout scaling across different screen sizes.

---

## Deployment

This website is deployed using **GitHub Pages**.

To deploy updates:

1. Push changes to the `main` branch.
2. GitHub Pages automatically rebuilds and publishes the site.
